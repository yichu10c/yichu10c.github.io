About

Software engineer building AI-powered systems at scale

I'm a software engineer at Nationwide Insurance, where I work on backend systems powering real-world financial products. I specialize in Python, FastAPI, and PostgreSQL — building APIs that handle heavy loads, background tasks with Celery, and integrating AI features into production systems.

Outside of work, I'm building on my own projects — exploring AI agent frameworks, contributing to open source, and building tools that automate my own workflow. I'm particularly interested in the intersection of AI and practical software — building things like resume parsers, job-matching engines, and intelligent automation.

This portfolio is self-hosted on AWS with automated CI/CD deployment pipelines, managed by an AI agent I built and configured myself.
