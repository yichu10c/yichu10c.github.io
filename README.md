# Yihone Chu — Portfolio

Personal portfolio website showcasing projects, skills, and accomplishments.

## Quick Start

Open `index.html` in any modern browser — it's a fully self-contained static site.

## Tech Stack

- Pure HTML + CSS + vanilla JavaScript (no build step, no dependencies)
- Google Fonts: Inter + JetBrains Mono
- CSS custom properties, glassmorphism, scroll reveal animations
- Fully responsive

## Sections

- **Hero** — name, tagline, tech stack tags
- **About** — bio, animated avatar, stats
- **Skills** — 5 skill cards with tech tags
- **Projects** — featured projects with live/planned badges
- **LeetCode** — progress tracker
- **Contact** — social links + contact form

## Customizing

All content is in `index.html` — edit the text directly to update projects, skills, links, etc.

## Deploy

Free hosting options:
- **GitHub Pages** — push to a `gh-pages` branch
- **Cloudflare Pages** — drag and drop
- **Netlify** — drag and drop the folder
